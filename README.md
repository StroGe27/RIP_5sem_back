# Аренда виртуальных машин

Система для автоматизации аренды виртуальных машин, которая будет состоять из веб-приложения, веб-сервиса, удаленного сервиса определяющего доступность домена и мобильного приложения.

Основное назначение разработанной системы заключается в автоматизации и оптимизации процесса аренды виртуальных машин. Система позволит быстро находить информацию о всех интересующих тарифах на аренду. Также она позволит программистам – быстро и легко создавать заявки на аренду виртуальных машин, а администраторам – принимать / отклонять их, тем самым решив проблему хостинга электронных приложений в интернете.
